Estrutura básica do HTML

Dentro do seu aplicativo é importante criar uma pasta para armazenar os seus projetos, para melhor organização e funcionalidade.

O HTML se consiste em 3 partes, na sua primeira tag utilizamos o "html" para definir o corpo do HTML e para finalizar o corpo é necessário colocar a barra para definir o final da tag tendo a sua funcionalidade por exemplo: "/html".

Dentro do HTML temos duas partes sendo esta o "head" e o "body", tudo que estiver no head é algo que vai ser carregado antes de aparecer para o usuário e o body é o conteúdo que aparece para o usuário.

Dentro do head temos a tag title onde fica o título do seu site.

Falando sobre Tags

Temos a tag 'i' que significa deixar o texto em italico; Temos a tag 'strong' que deixa o texto em negrito;

E também possui as tags que não tem o fechamento, uma delas é a tag 'input' e 'img'

As tags tem atributos que podem ser inseridos, por exemplo a tag 'input type=""/> onde o type é um atributo dessa tag e tem vários valores que podem ser atribuídos e alterados

Atributos das Tags

Atributos nada mais é do que as propriedades que uma tag podem ter, tem atributos gerais e outros mais específicos das próprias tags. Possui a tag id dando um identificador, a tag style que é utilizada mais no CSS e a tag class que serve para facilitar a formatação de conteúdos principais

A tag input já tem mais as propriedades específicas, onde o type define vários valores e particularidades para campo

E a tag 'img' permite colocar uma imagem diretamente ao seu site

Textos

A primeira tag que vamos aprender é a tag H, onde o H significa head. Os H1 tem a sua importância definida pela ordem dos números. O h's servem para títulos, já a tag 'p' serve para parágrafos. O h1 serve como um título principal da sua página e o h2 já fica como sub-título.

Caso você queira utilizar uma citação, possui a tag 'blockquote' para deixar como uma citação de alguma fala ou alguma referência externa do seu site

Você pode estilizar o seu próprio texto com as tags, além das tags 'i', 'strong', também temos a tag 'u', 'mark','sup','sub'

Listas ordenadas e não ordenadas

Existem dois tipos de listas, que possuem seus padrões e que podem ser alterados via CSS.

A tag 'ol' é a lista ordenada, os items tem que ser colocados de forma organizada sendo as tags filhas precisando a tag 'li' sendo o list item. A lista ordenada é colocada como númerica, sendo em ordem númerica 1,2,3....

E a tag 'ul' é a lista não ordenada, onde ao invés de ser números é colocada como bolinha, que não é organizada por números e essa tag é utilizada bastante para o menu

Links

(Voltar)

Para fazer o link em nosso HTML, é utilizada a tag 'a' que significa anchor algo que âncora algo.

Na tag temos o href="" onde você pega de forma fora da página e coloca linkada em seu site, essa tag é necessário fechar e pode ser colocada um texto como um título para o seu link

Para o link abrir em outra aba é o target="" e colocamos o _blank para abrir o link em outra aba sem ser na sua própria página

O title no serve para colocar um balão descritivo sobre o seu link

Pode ser feito a navegação dentro do seu próprio site, criando um link entre os arquivos
